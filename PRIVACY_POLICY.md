# Privacy Policy

**Last Updated: November 24, 2025**

## Introduction

This Privacy Policy explains how MISAKI Bot ("the Bot", "we", "us", or "our") collects, uses, and protects your information when you use our Discord bot.

## Information We Collect

### Automatically Collected Information

When you use MISAKI Bot, we automatically collect:

1. **Server Information**
   - Server ID (Guild ID)
   - Server name
   - Voice channel IDs
   - Text channel IDs

2. **User Information**
   - User ID
   - Username and display name
   - Voice channel activity (only when using music features)

3. **Usage Data**
   - Commands executed
   - Music playback history (stored temporarily per server)
   - Bot settings (prefix, 24/7 mode)

### Data Storage

We store the following data in our database:

1. **Server Configuration**
   - Custom command prefix (per server)
   - 24/7 mode settings (enabled/disabled, channel ID)

2. **Temporary Music Data**
   - Current playback queue (cleared when bot disconnects)
   - Playback history (last 10 tracks, stored in memory only)
   - Current player state

3. **Favorites** (if using favorites feature)
   - User ID
   - Saved track information

## How We Use Your Information

We use collected information to:
- Provide music playback functionality
- Remember server-specific settings
- Maintain independent queues per server
- Enable 24/7 auto-join features
- Improve Bot performance and features
- Debug issues and errors

## Data Retention

- **Server Settings**: Stored until server removes the Bot or settings are changed
- **Music Queue**: Cleared when Bot disconnects from voice
- **Playback History**: Stored in memory, cleared on Bot restart
- **24/7 Settings**: Stored until manually disabled
- **Invalid Server Data**: Automatically cleaned up during Bot startup

## Data Sharing

We DO NOT:
- Sell your data to third parties
- Share your data with advertisers
- Share server data between different servers
- Store personal messages or conversation content
- Track users across different servers

We MAY share data:
- If required by law or legal process
- To protect our rights or property
- With your explicit consent

## Third-Party Services

MISAKI Bot uses the following third-party services:

1. **Discord API**
   - For all Bot functionality
   - Subject to [Discord's Privacy Policy](https://discord.com/privacy)

2. **Wavelink/Lavalink**
   - For music streaming
   - Music is streamed in real-time, not stored

3. **Music Sources** (YouTube, Spotify, JioSaavn, SoundCloud)
   - Metadata retrieval only
   - Subject to respective platform privacy policies

## Data Security

We implement reasonable security measures to protect your data:
- Data is stored locally in secure databases
- No sensitive personal information is collected
- Server data is isolated (multi-server support)
- Regular cleanup of invalid entries

## Your Rights

You have the right to:
- Request information about data we store
- Request deletion of your server's data (remove the Bot)
- Change or reset server settings at any time
- Disable data collection by not using the Bot

### How to Exercise Your Rights

- **Delete Server Data**: Remove the Bot from your server or use admin commands
- **Reset Settings**: Use `/prefix` command or disable 24/7 mode
- **Clear History**: Use `/clearhistory` command

## Children's Privacy

MISAKI Bot does not knowingly collect information from users under 13 years of age. The Bot is intended for use in accordance with Discord's Terms of Service, which requires users to be at least 13 years old.

## Multi-Server Privacy

Each server's data is completely separate:
- Queues are independent per server
- Settings are isolated per server
- Playback history is not shared between servers
- Each server has its own database entries

## Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Changes will be effective immediately upon posting. Your continued use of the Bot constitutes acceptance of the updated Privacy Policy.

## Data Deletion

To request complete data deletion:
1. Remove the Bot from your server (automatic cleanup)
2. Contact the developer for manual data removal
3. Data associated with your server will be deleted

## Contact Information

If you have questions about this Privacy Policy or your data, please contact the Bot developer through Discord.

## Consent

By using MISAKI Bot, you consent to this Privacy Policy and agree to its terms.
